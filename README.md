#### I created this simple Login Form with interactive elements and restrictions so that users will know what to input
#### I use the attribute `pattern="^[A-Za-z\d@$!%*?&]{10,}$"`, on `<input type="passwords">` which restricts users from inputting on passwords with unnecessary characters.
#### This project teaches me how to give proper restrictions to users which gives them an idea what to input in a form.
